Make WordPress

新米のデベロッパー、細部にこだわるデザイナー、協力し合うことが好きな方など、私たちは常に WordPress の改善の協力者を探しています。

WordPress に参加するには、ここから始めましょう。それぞれのコントリビューターグループのブログでは、ニュース、近日のイベントなどをお知らせします。 Codex にも、「WordPress への協力（日本語）」というページがあります。

WordPress に参加する方法は、たくさんあります：

コア

コアチームは、 WordPress 本体を作っています。経験を積んだ PHP 開発者でも、コードを学んだばかりの方でも、歓迎します。コーディングや、バグフィックス、意思決定のディベート、開発の改善などでご参加下さい。

コアについて詳しく知る（英語） »

UI

UI グループは、ユーザーインターフェースのデザインと開発に焦点を当てます。デザイナーや、 UX デザイナーのホームのようなものです。モックアップ、デザインやユーザーテストの定期的なディスカッションがあります

UIについて詳しく知る（英語） »

モバイル

モバイルチームは iOS や、Android、Windows Phone、Blackberry のアプリをつくります。 Java、Objective-C、C# のスキルを貸してください。さらに、デザイナー、 UX のエキスパート、そして全てのデバイスでスムーズな体験を提供するための動作テスターも必要です。

モバイルについて詳しく知る（英語） »

アクセシビリティ

アクセシビリティグループは、 WordPress プロジェクト全体にて、アクセシビリティの知識を提供します。 WordPress 本体や、すべての WordPress のリソースが、アクセシブルであることを確認します。

アクセシビリティについて詳しく知る（英語） »

多言語対応

WordPress は世界中のさまざまな言語で使われています。ポリグロッツ（多言語話者）の方は、 WordPress を様々な言語に翻訳してください。また、翻訳のためのツールをつくることも役立ちます。

多言語対応について詳しく知る（英語） »

サポート

サポートフォーラムや、 Slack での質問に答えることは、貢献を始める最も簡単な方法です。誰しも、何かしらの質問には答えることができます！ブログでは、サポートに関するイシューを議論します。

サポートについて詳しく知る（英語） »

ドキュメンテーション

良いドキュメンテーションは、課題を自力で解決するのを促します。ドキュメンテーションチームは、ドキュメンテーションをつくる責任者として、寄せられる投稿をチェックしています。ブログでは、チームの最新のプロジェクトについて議論します。

ドキュメンテーションについて詳しく知る（英語） »

テーマ

テーマレビューチームは、 WordPress のテーマレポジトリに応募されたそれぞれのテーマをレビューし、承認します。テーマのレビューをすることで、テーマ開発のスキルが鋭くなります。ブログから参加できます。

テーマについて詳しく知る（英語） »

プラグイン

プラグイン開発者なら、最新のアップデートについていくためにプラグインレビューチームのブログを購読し、リソースを見つけ、プラグイン開発にまつわる課題について学んで下さい。

プラグインについて詳しく知る（英語） »

コミュニティ

勉強会や WordCamp の運営に興味があるなら、コミュニティブログを読むところからはじめましょう。イベントのサポートや、ボランティア活動やトレーニングをするためのプログラム作成、コミュニティの総合的なサポートをしています。

コミュニティについて詳しく知る（英語） »

メタ

メタチームは WordPress.org をつくり、サポートをし、すべてのコントリビューターグループが使うためのツールをつくります。もし WordPress.org をもっと良くしたいなら、メタブログの購読登録をしてください。

メタについて詳しく知る（英語） »

トレーニング

トレーニングチームは、インストラクターが実際のワークショップで使うための、ダウンロード可能なレッスンプランや関連資料をつくります。 WordPress を使って何かを作ることを教えている方は、少し手を止めて私たちのチームに入って下さい！

トレーニングについて詳しく知る（英語） »
